# Mediatek
Mediatek est un projet en Symfony permettant de générer une médiateque qui prend en charge Films, Séries, Animations et Mangas. L'utilisateur peut ajouter et modifier son contenu et ses liens

<img src="http://100.27.5.27/images/media.png"
alt="Image du projet Mediatek"
style="float: left; margin-right: 10px;" />

## Premiers pas

- Clonez le projet
- Faite un "composer install" dans la racine du projet
- Creer un fichier .env.local avec la configuration de votre base de donnée
- Procéder à la migration
